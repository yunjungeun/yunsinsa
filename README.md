# yunsinsa (E-commerce)
온라인 쇼핑몰의 주요기능인 주문/결제 기능

<h4>개인프로젝트를 만든 목적</h4>
팀프로젝트에서 맡지 못한 기능에 대한 아쉬움과<br>
팀프로젝트에서 하지 않은 기술을 사용하고 싶어 개인프로젝트를 시작함.

<h4>일정</h4>
2023.10.15~2023.11.30 <br>
참여 100%

# 사용언어 및 도구
JAVA, JPA, MySQL <br>
spring boot, gradle, docker, postman, github
